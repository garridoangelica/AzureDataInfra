# AzureDataInfra
Repository for deployments of Azure Data resources through IaC.

## Fabric CLI Deployment Framework
Check out the [Fabric CLI deployment guide](Fabric/CLIDeployment/README.md) for a comprehensive solution to automate Microsoft Fabric workspace deployments using the Fabric CLI.

## Spark Security Scan Pipeline for Microsoft Fabric
Explore the [Spark Security Pipeline](Fabric/SparkHistoryNew/README.md) - a tool for extracting and analyzing Spark session logs from Microsoft Fabric workspaces to identify security concerns, external connections, and potential data exfiltration risks. This pipeline helps security teams monitor notebook activities, detect unauthorized external connections, and generate detailed security reports.
